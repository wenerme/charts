# Helm Charts

```bash
helm repo add wener https://wenerme.github.io/charts
helm search repo wener/
```

## Mirror charts
Collect a lot offcial charts for easy to find and use in one repo.

* Auto update based on Github Action

## DEV

```bash
# Only clone charts

```
